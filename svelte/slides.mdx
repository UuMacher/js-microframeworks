# Svelte 3.0

<blockquote cite="https://svelte.dev/">Cybernetically enhanced web apps</blockquote>

---

## Features

- Svelte = Compiler
- _No runtime_
- Write less code
- Truly reactive
- No virtual DOM
- Surgically updates the DOM
- Vanilla JS
- *fast* (faster than Angular, Ember, React, Preact, Riot, Mithrill, ...)

---

## Principles

- Declarative components (.svelte, single file)
    - JavaScript, HTML, CSS
    - Special syntax for reactive statements
- Svelte converts components into plain JavaScript

## Getting started
### Use degit / svelte-template
```
npx degit sveltejs/template my-svelte-project
cd my-svelte-project
npm install
npm run dev
```
